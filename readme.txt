== Description ==

Social Widget Plugin lets you have a widget that contains all major social network icons. Customisable colours, drag and drop. Easy to use.


== Installation ==

1. Upload the entire Sitemile Social Widget folder to the /wp-content/plugins/ directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. In your Widgets menu, simply drag the widget labeled "SiteMile Social Widget" into a widget area.
1. Configure the widget by choosing a title, icon size and color, and the URLs to your various social profiles.

== Frequently Asked Questions ==

= How can I add an icon? =

You can add an icon by editing the main php file of the plugin. Simply tie to the filter, or add the filter code into your functions.php
